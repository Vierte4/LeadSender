
Leads_Sender это программа, которая каждые 25-35 минут проверяет наличие новых лидов по всем формам клиента и, в случае
нахождения таковых, отправляет их клиенту

Программа состоит из 3х частей:
1. Бот в телеграме, который отвечает за общение с клиентом. Возможности бота:

        Возможности клиента:
        - Включить/выключить бота для себя
        - Получать лиды

        Возможности админа:
        - Включить/выключить бота для любого клиента
        - Получать сообщения о включении/выключении бота для любого клиента
        - Запросить информацию о состоянии бота для всех клиентов

        Особенности работы:
        - Клиент подключается вручную разработчиком, который вносит все его данные в базу.
        - В случае выключения бота для клиента, он перестает скачивать и отправлять его лиды. При включении и выключении
          бота сообщение об этом получает как клиент, так и админ.
        - Возможности админа:

2. База данных. Содержит только информацию по клиентам, которая необходима для работы всей программы. Представлена в
   виде словаря:

   {ID_клиента: название_рабочей_страницы: название_формы: количество_лидов_формы}

3. Скачивание лидов клиентов. Скачивание лидов происходит раз в 25-35 минут, поочередно для всех клиентов. Логика:
